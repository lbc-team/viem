# showCallsStatus

请求钱包显示通过 `showCalls` 发送的调用批次的信息。

[了解更多。](https://github.com/ethereum/EIPs/blob/1663ea2e7a683285f977eda51c32cec86553f585/EIPS/eip-5792.md#wallet_showcallsstatus)

:::warning[警告]
这是一个实验性操作，大多数钱包不支持。建议在生产环境中使用时有一个后备机制。
:::

## 用法

:::code-group

```ts twoslash [example.ts]
import { walletClient } from './config'
 
await walletClient.showCallsStatus({ // [!code focus:99]
  id: '0x1234567890abcdef',
})
```

```ts twoslash [config.ts] filename="config.ts"
import 'viem/window'
// ---cut---
import { createWalletClient, custom } from 'viem'
import { mainnet } from 'viem/chains'
import { walletActionsEip5792 } from 'viem/experimental'

export const walletClient = createWalletClient({
  chain: mainnet,
  transport: custom(window.ethereum!),
}).extend(walletActionsEip5792())

export const [account] = await walletClient.getAddresses()
```

:::

## 返回

`void`

## 参数

### id

- **类型:** `string`

调用批次的标识符。
