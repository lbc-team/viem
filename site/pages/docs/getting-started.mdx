# 入门 [仅需几行代码即可开始使用 viem。]

## 概述

viem 是一个用于以太坊的 TypeScript 接口，提供与以太坊交互的低级无状态原语。viem 专注于开发者体验、稳定性、包大小和性能：

- **开发者体验** 自动的 [类型安全和推断](/docs/typescript)、全面的文档、可组合的 API。
- **稳定性** 测试套件在分叉的以太坊网络上运行，完整的 [测试覆盖率](https://app.codecov.io/gh/wevm/viem)。
- **包大小** 可树摇的轻量级模块。
- **性能** 优化的编码/解析，仅在必要时执行异步任务。

你可以在 [为什么选择 viem](/docs/introduction) 部分了解更多关于该项目的背景信息。

## 安装

## 快速开始

### 1. 设置你的客户端和传输

首先，使用所需的 [传输](/docs/clients/intro) 和 [链](/docs/chains/introduction) 设置你的 [客户端](/docs/clients/intro)。

```ts twoslash
import { createPublicClient, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = createPublicClient({ // [!code focus]
  chain: mainnet, // [!code focus]
  transport: http(), // [!code focus]
}) // [!code focus]
```

:::info
在生产应用中，强烈建议传入你的认证 RPC 提供者 URL（Alchemy、Infura、Ankr 等）。如果未提供 URL，viem 将默认使用公共 RPC 提供者。 [阅读更多](/docs/clients/transports/http#usage)。
:::

### 2. 消费操作

现在你已经设置了客户端，可以与以太坊交互并消费 [操作](/docs/actions/public/introduction)！

```ts twoslash 
import { createPublicClient, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = createPublicClient({
  chain: mainnet,
  transport: http(),
})

const blockNumber = await client.getBlockNumber() // [!code focus]
```
