# 概述

```ts twoslash
// 1. 导入模块。
import { createPublicClient, http } from 'viem'
import { mainnet } from 'viem/chains'

// 2. 使用所需的链和传输设置客户端。
const client = createPublicClient({
  chain: mainnet,
  transport: http(),
})

// 3. 执行一个操作！
const blockNumber = await client.getBlockNumber()
```

# 特性

viem 开箱即用地支持以下所有功能：

- 对 [JSON-RPC API](https://ethereum.org/en/developers/docs/apis/json-rpc/) 的抽象，使你的生活更轻松
- 与 [智能合约](https://ethereum.org/en/glossary/#smart-contract) 交互的第一类 API
- 与官方 [以太坊术语](https://ethereum.org/en/glossary/) 密切对齐的语言
- 导入你的浏览器扩展、WalletConnect 或私钥钱包
- 浏览器原生 [BigInt](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/BigInt)，而不是大型 BigNumber 库
- 用于处理 [ABI](https://ethereum.org/en/glossary/#abi) 的工具（编码/解码/检查）
- TypeScript 准备就绪（从 ABI 和 EIP-712 类型数据 [推断类型](/docs/typescript)）
- 对 [Anvil](https://book.getfoundry.sh/)、[Hardhat](https://hardhat.org/) 和 [Ganache](https://trufflesuite.com/ganache/) 的第一类支持
- 针对 [分叉](https://ethereum.org/en/glossary/#fork) 以太坊网络运行的测试套件
